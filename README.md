 # 🎓 StudentManager — 学生管理系统

  基于 .NET 10 的控制台学生信息管理系统，使用结构体数组实现学生信息的增删改查。

  ## ✨ 功能

  | 功能 | 说明 |
  |------|------|
  | 添加学生 | 录入学号、姓名、年龄、班级，自动检测学号重复 |
  | 显示全部 | 列表展示所有学生信息 |
  | 查找学生 | 按学号精准查找 |
  | 删除学生 | 按学号删除，自动整理数组 |
  | 修改学生 | 按学号定位，修改全部字段 |

  ## 数据结构

  ```csharp
  struct Student {
      string id;     // 学号（唯一）
      string name;   // 姓名
      int age;       // 年龄
      int classname; // 班级
  }

  - 固定大小数组（最大50人），维护 count 追踪实际数量
  - 删除时元素前移压缩数组

  运行

  cd studentmanager
  dotnet run

  🛠 技术栈

  - C# (.NET 10)
  - 内存数组存储（无数据库依赖）
